### 予約語の記載方法
タグとして予約されている語彙と、翻訳されず原語のまま使用されている語彙が区別できなくなるため、予約語はフォントと色を変更する。

### 定義語の記載方法
Object、Arrayなど、文書内で定義される語彙は、原語のまま利用する。例の中にも記載されるためで、翻訳することにより対応関係が一見してわかりにくくなるため。
